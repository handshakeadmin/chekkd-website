**Solution:**
1. Check meta tag in live HTML source (view-source:https://chekkd.com)
2. Use GSC URL Inspection tool
3. Request indexing again if needed

### Issue: Logo not showing after 2 weeks
**Solution:**
1. Verify logo loads: https://chekkd.com/images/chekkd-logo-1200x1200.png

3. Wait another week (can take 3-4 weeks sometimes)

### Issue: Impressions dropping
**Solution:**
1. Check GSC Coverage for errors
2. Verify sitemap still submitted
3. Check for manual actions in GSC

**Solution:**
1. Delete sitemap from GSC
2. Re-add with: https://www.chekkd.com/sitemap.xml
